# Epi Zafferano Frontend

## Avvio

-   installare le dipendenze:

    ```
    npm i
    ```

-   copiare il file .env in .env.local e inserire i valori necessari:

    ```
    cp .env .env.local
    ```

    oppure copiarlo e rinominarlo manualmente

-   avviare il server di sviluppo:
    ```
    npm run dev
    ```
